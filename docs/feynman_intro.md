## 1. 一句话介绍

这个项目想回答一个问题：

> 能不能用一种更接近生物神经系统的学习规则，产生和深度学习标准反向传播相似的权重更新方向？

再说得更白一点：

> 深度学习里的 BP 很有效，但不太像大脑。这个项目尝试检查一种更像生物神经元工作方式的规则，能不能在 MNIST 手写数字识别任务中“学得像 BP”。

---

## 2. 先理解一个比喻

想象你在教一个学生认数字。

你给他看一张图片，问：

```text
这是 3 吗？
```

学生回答错了。你需要告诉他哪里错了，然后让他调整自己的判断方式。

在普通深度学习里，这个“告诉他哪里错了”的过程叫：

```text
反向传播 Backpropagation，简称 BP
```

BP 会非常精确地计算：

```text
每一层神经网络里的每一个连接，应该往哪个方向改，改多少。
```

这很强，但也有一个问题：

> BP 的计算方式很像数学工程系统，不太像真实大脑里的神经元和突触。

这个项目要做的事情就是：

> 拿一个更有生物味道的规则，看看它给出的“修改方向”和 BP 给出的“修改方向”有多像。

---

## 3. 这个项目解决了什么问题？

它不是直接解决“让 MNIST 准确率最高”的问题。

它真正解决的是一个研究验证问题：

> 如果我们提出一种更符合生物机制的学习规则，怎么判断它是否接近标准深度学习中的 BP？

所以项目的核心不是只看 accuracy，而是看：

```text
BP 想怎么改权重
生物规则想怎么改权重
这两个“修改方向”之间差多少
```

这个差距用一个很直观的指标表示：

```text
夹角 angle
```

---

## 4. 为什么用“夹角”？

假设有两个人同时推一张桌子。

- 如果两个人往同一个方向推，夹角接近 `0` 度。
- 如果一个往东推，一个往北推，夹角接近 `90` 度。
- 如果一个往东推，一个往西推，夹角接近 `180` 度。

神经网络的权重更新也是类似的。

每一种学习规则都会给出一个“权重应该怎么改”的方向：

```text
BP 更新方向
生物规则更新方向
```

我们把这两个方向拿来比较：

| 夹角 | 含义 |
| --- | --- |
| 接近 `0` 度 | 两种规则几乎在做同一件事 |
| 接近 `90` 度 | 两种规则关系不强 |
| 接近 `180` 度 | 两种规则方向几乎相反 |

所以这个项目画的趋势图，本质上是在问：

> 随着训练进行，生物规则是不是越来越像 BP？

---

## 5. 项目里两种学习规则分别是什么？

### 5.1 标准 BP

标准 BP 是深度学习里的经典算法。

它会根据 loss 精确计算梯度，然后更新权重：

```text
Delta W = -learning_rate * gradient
```

通俗解释：

> 哪个连接导致错误变大，就把它往减少错误的方向调。

### 5.2 生物合理规则

这个项目中的生物合理规则有几个特点：

1. 神经元激活值保持为正。
2. 误差信号不是普通 BP 那样传播。
3. 误差信号会被 `zeta` 函数截断或压缩到较小范围。
4. 权重更新不仅看误差，也看当前神经元活动。

通俗解释：

> 它更像是在模拟神经元局部活动和误差信号共同影响突触变化的过程。

---

## 6. 为什么要求激活值为正？

真实神经元的放电频率通常不是负数。

它可以低，可以高，但不会出现：

```text
放电频率 = -3
```

所以这个项目让神经网络每一层的激活值都为正：

```text
activation = softplus(x) + eps
```

可以简单理解为：

> 网络里每个神经元输出的是一种“活动强度”，活动强度始终大于 0。

这让模型更接近生物直觉。

---

## 7. 为什么需要 zeta？

生物系统里的信号通常不会无限大。

如果误差信号特别大，真实系统往往会有某种饱和、截断或压缩机制。

所以项目里引入：

```text
zeta
```

它的作用是把误差信号限制在：

```text
(-0.5, 0.5)
```

通俗解释：

> 不管原始误差多夸张，进入学习规则之前都先压到一个温和范围。

默认实现是：

```text
zeta(x) = 0.5 * tanh(x)
```

---

## 8. 实验具体在做什么？

项目使用 MNIST 数据集，也就是手写数字图片。

网络结构是：

```text
784 -> 500 -> 500 -> 500 -> 10
```

含义：

- 输入是 `28 x 28 = 784` 个像素。
- 中间有 3 个隐藏层。
- 每个隐藏层有 500 个神经元。
- 输出有 10 个类别，对应数字 `0` 到 `9`。

每训练一个 batch，项目都会做这几件事：

```text
1. 用当前网络看一批 MNIST 图片
2. 计算当前预测错误 loss
3. 计算标准 BP 会怎么改权重
4. 计算生物规则会怎么改权重
5. 比较两个权重更新方向的夹角
6. 按你选择的 train-rule 真正更新网络
7. 把 loss、accuracy、夹角记录下来
```

---

## 9. 输出结果怎么看？

运行实验后会得到两个主要文件：

```text
outputs/angle_metrics.csv
outputs/angle_trend.png
```

### CSV 文件

CSV 里每一行对应一个 epoch。

重要字段：

| 字段 | 含义 |
| --- | --- |
| `train_loss` | 训练损失 |
| `train_accuracy` | 训练准确率 |
| `mean_angle_degrees` | 所有层平均夹角 |
| `layer_1_angle_degrees` | 第一层夹角 |
| `layer_2_angle_degrees` | 第二层夹角 |
| `layer_3_angle_degrees` | 第三层夹角 |
| `layer_4_angle_degrees` | 第四层夹角 |

### 趋势图

趋势图最重要的是看：

```text
mean_angle_degrees 是否下降
```

如果下降，说明：

> 随着训练进行，生物规则产生的更新方向越来越接近 BP。

如果不下降，说明：

> 这个生物规则在当前设置下，并没有明显变得更像 BP。

## 10. 这个项目有哪些亮点？

### 🧪 亮点 1：比较的是“学习方向”，不是只看准确率

很多实验只看最后 accuracy 高不高。

但这个项目更进一步，直接比较：

```text
两种学习规则到底是不是在往相似方向更新权重
```

这对研究一种新学习规则是否接近 BP 很重要。

### 🧬 亮点 2：把生物约束写进了网络

项目不是普通 MLP 直接套公式，而是加入了：

- 正值激活
- 误差信号压缩
- 活动依赖更新

这些设计都更接近生物系统中的直觉。

### 📐 亮点 3：指标直观

夹角是一个很容易理解的指标。

它不需要你先理解复杂梯度推导，也能直观看出：

```text
两个学习规则像不像
```

### 🧰 亮点 4：实验可复现

项目提供了一键脚本：

```bash
./scripts/run_experiment.sh --smoke
./scripts/run_experiment.sh --epochs 20
```

脚本会自动：

- 创建环境
- 安装依赖
- 跑测试
- 运行实验
- 生成图表

### 📝 亮点 5：结构清晰，方便继续研究

代码拆成了几个清楚的部分：

| 文件 | 作用 |
| --- | --- |
| `model.py` | 定义正值激活神经网络 |
| `learning_rules.py` | 定义 BP、生物规则和夹角计算 |
| `train_mnist.py` | 跑 MNIST 实验 |
| `run_experiment.sh` | 一键运行 |

你以后要改 `zeta` 或 `f(epsilon, y)`，主要改 `learning_rules.py` 即可。

---

## 11. 它不是什么？

为了避免误解，也要说清楚这个项目目前不是什么。

它目前不是：

- 一个追求 MNIST 最高准确率的项目。
- 一个完整模拟真实大脑的项目。
- 一个已经证明生物规则一定优于 BP 的项目。
- 一个包含大量数据集和模型结构的完整 benchmark。

它更像是：

> 一个研究型实验框架，用来检验某个生物合理学习规则与 BP 更新方向之间的关系。

---

## 12. 最推荐怎么开始？

第一步，先跑：

```bash
./scripts/run_experiment.sh --smoke
```

确认项目能跑。

第二步，跑正式实验：

```bash
./scripts/run_experiment.sh --epochs 20 --open
```

第三步，对比 BP 训练轨迹：

```bash
./scripts/run_experiment.sh \
  --epochs 20 \
  --train-rule bp \
  --output-csv outputs/bp_metrics.csv \
  --output-plot outputs/bp_angle_trend.png
```

第四步，比较两张图：

```text
outputs/angle_trend.png
outputs/bp_angle_trend.png
```

看看生物规则在不同训练轨迹上，与 BP 的夹角趋势是否一致。

---

## 13. 如果用一句更生活化的话总结

可以这样理解：

> BP 像一个数学老师，告诉神经网络每个连接应该怎么改。这个项目提出了一个更像生物神经系统的老师，然后检查这个新老师给出的修改建议，和 BP 老师给出的建议到底有多接近。

如果两个老师给出的建议方向越来越像，那说明这个生物规则可能抓住了 BP 中某些关键的学习结构。
